# Iris Insight Classifier 

A machine learning project that predicts the species of Iris flowers using their measurements.

## Overview
This project uses the famous Iris dataset to classify flowers into three categories:
- Setosa
- Versicolor
- Virginica

The model learns patterns from flower features and predicts the species accurately.

## Features Used
The dataset includes:

- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

These features are used for prediction.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Project Workflow
1. Load the dataset
2. Explore the data
3. Visualize flower patterns
4. Split data into training and testing sets
5. Train the machine learning model
6. Test the model performance
7. Predict flower species

## Model Used
This project uses the Random Forest Classifier for classification.

## Accuracy
The model gives high accuracy on the Iris dataset.
